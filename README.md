# pychat 
Pychat is a Python QT5 chatbot  designed to assist users with their queries. 
### Installation 
To install the dependencies required for this project, please follow the steps below: 
``` bash  
    # Rename the .env.example file to .env
    cp .env.example .env
    
    # Add your OpenAI API key to the OPENAI_KEY environment variable in the .env file
    OPENAI_KEY=your_api_key_here
    
    # Install the dependencies listed in the requirements.txt file using pip
    pip install -r requirements.txt
```
   
Once you have completed these steps, you can run the pychat chatbot by executing the following command: 
  
    python pychat.py
   
Enjoy chatting with the pychat bot!
